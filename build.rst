build
=======


Pre-installed
----------------

* docutils
* rst2pdf
* scons

for debian-based release:

  .. code-block:: shell
    
    sudo apt-get install rst2pdf scons python-docutils



building
----------

building the documents, please follow cmd as below:

  .. code-block:: shell
    
    scons 

cleaning the documents:

  .. code-block:: shell
    
    scons -c
